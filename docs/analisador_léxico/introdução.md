## O que é um analisador léxico?

O **analisador léxico** (também chamado de *lexer* ou *scanner*) é a fase que lê o fluxo de caracteres de um programa fonte e agrupa esses caracteres em sequências significativas chamadas *Lexemas*. Para cada lexema, o analisador produz um *Token* como saída para o analisador sintático.
